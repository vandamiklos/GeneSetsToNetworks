# GeneSetsToNetworks

A collection of functions to convert gene-set enrichment analysis results to a network graph.

## Install

```
library(devtools)
install_github("vandamiklos/GeneSetsToNetworks")
```

## Prerequisites:

- library(devtools)
- library(tidyverse)
- library(igraph)
- library(ggraph)
